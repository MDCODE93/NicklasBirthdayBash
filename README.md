# Personal Website Template

A clean, responsive personal website template built with HTML and CSS, perfect for showcasing your professional profile on GitHub Pages.

## Features

- **Responsive Design**: Looks great on desktop, tablet, and mobile devices
- **Professional Layout**: Sections for About, Education, Experience, Recommendations, and Contact
- **Easy Customization**: Simple HTML and CSS structure
- **Free Hosting**: Deploy for free on GitHub Pages

## Quick Start

1. **Create a GitHub Repository**:
   - Go to GitHub and create a new repository
   - Name it `yourusername.github.io` (replace with your GitHub username) for a custom domain, or any name for `yourusername.github.io/repository-name`
   - Clone the repository to your local machine

2. **Add the Template Files**:
   - Copy `index.html` and `style.css` to your repository
   - Or download them from this template

3. **Customize Your Content**:
   - Open `index.html` in any text editor
   - Replace placeholder text with your information:
     - Your name and photo
     - Professional summary
     - Education details
     - Work experience
     - Recommendations/testimonials
     - Contact information

4. **Add Your Photo**:
   - Replace the placeholder image URL with your photo
   - Upload your photo to the repository (e.g., in an `images/` folder) or use an external URL

5. **Deploy to GitHub Pages**:
   - Commit and push your changes to GitHub
   - Go to your repository Settings
   - Scroll to "Pages" section
   - Select "Deploy from a branch"
   - Choose "main" branch and "/ (root)" folder
   - Click Save
   - Your site will be live at `https://yourusername.github.io` (or the repository URL)

## Customization

- **Styling**: Edit `style.css` to change colors, fonts, and layout
- **Content**: Modify `index.html` to add/remove sections or change structure
- **Photos**: Add an `images/` folder for local images

## Adding to LinkedIn

Once your site is live:
1. Copy the URL (e.g., `https://yourusername.github.io`)
2. Go to your LinkedIn profile
3. Add it to:
   - **Featured section**: Add a new featured item with the link
   - **Contact info**: Add "Website" in the contact section
   - **About section**: Mention it in your summary

## Troubleshooting

- **Site not loading**: Wait 5-10 minutes after enabling Pages
- **Changes not showing**: Hard refresh your browser (Ctrl+F5)
- **Custom domain**: If using a custom domain, add a `CNAME` file
